# 🚀 Otimizador de Escritório 300%

*Transforme o caos em organização com apenas 2 cliques.*

Este software foi desenvolvido para profissionais e escritórios que lidam com um grande volume de arquivos diários e perdem tempo precioso movendo itens manualmente para pastas.

## 🛠 O que ele faz?
* *Organização Inteligente:* Separa automaticamente Documentos, Imagens, Planilhas e Vídeos.
* *Modo Reunião:* Cria pastas específicas para organizar materiais de encontros em segundos.
* *Prioridade Urgente:* Botão de acesso rápido para mover arquivos críticos para uma área de foco.
* *Log de Auditoria:* Gera um arquivo .txt automático com o histórico de tudo o que foi organizado.

## 🧠 Por que usar?
Diferente de uma organização manual, o *Otimizador 300%* elimina o erro humano e garante que você encontre o que precisa, quando precisa. É tecnologia a serviço da produtividade.
